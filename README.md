# OpenWeatherMap + Wikipedia Automation Project

## Описание
Автоматизированные тесты для:
- Веб-сайта OpenWeatherMap (6 тестов)
- Мобильного приложения Wikipedia (4 теста)

## Технологии
- Java 11
- Selenium WebDriver 4.15.0
- Appium 8.5.1
- TestNG 7.8.0
- Maven
- WebDriverManager

## Требования
- JDK 11+
- Maven 3.6+
- Chrome/Firefox браузер
- Для мобильных тестов:
    - Node.js и Appium
    - Android Studio с эмулятором
    - APK файл Wikipedia
